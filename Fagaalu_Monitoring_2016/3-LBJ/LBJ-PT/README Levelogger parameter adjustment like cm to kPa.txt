Levelogger README

If your data is in some other unit than 'kPa'

Go to "Data Wizard" at the top of Solinst Levelogger program

Select "Advanced" radio button
click Next

Select "Parameter Adjustments Only"
click Next

Select the correct file
click Next

Check "Apply All" box

Use the drop-down menu for "Level" to select "kPa"
click Finish
